# linux-study

GPU 인프라 운영 실무에서 쓰는 리눅스 명령과 장애 대응 절차를
**"명령 / 언제 쓰나 / 무엇을 보나"** 형식으로 정리하는 학습 노트입니다.

## 구성
- [week1](./week1) — 프로세스·리소스·서비스 (top, free, systemctl, df/du, nvidia-smi)
- week2 — 로그·텍스트 처리 (grep, awk, sed, journalctl) *(진행 예정)*
- week3 — 네트워크 (ip, ss, tcpdump, InfiniBand)
- week4 — 쉘 스크립트 자동화

## 원칙
- 장애 대응을 "감"이 아닌 "절차"로 만든다
- 실무 수치·호스트 정보는 포함하지 않는다 (일반화된 지식만)
